      because it does not utilize tables, rows and columns to organize data

Q11 How does the value of _id get assigned to a document?

Sol: It is automatically generated as an ObjectId type value, You can select a non ObjectId type value when inserting a new document, as long as that value is unique to this collection.

Q12 Insert error, Select all true statements from the following list:
 Sol: MongoDB can store duplicate documents in the same collection, as long as their _id values are different, If a document is inserted without a provided _id value, then that field and value will be automatically generated for the inserted document before insertion
